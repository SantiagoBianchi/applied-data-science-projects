# Applied Data Science Projects

This repository contains applied data science projects focused on data analysis, statistical modeling, and machine learning using Python and R.

The projects emphasize end-to-end workflows, including data cleaning, exploratory data analysis (EDA), modeling, visualization, and interpretation of results on real-world datasets.

## Projects
- **Exploratory Data Analysis – Crime in Los Angeles**  
  Exploratory analysis of crime data focusing on demographics, temporal patterns, and night crime distribution.

- **Logistic Regression – Car Insurance Claims**  
  Feature-wise logistic regression analysis to identify key predictors of insurance claim outcomes.

- **Hypothesis Testing – Goals in World Cup Matches**  
  Statistical comparison of goal distributions in men's and women's World Cup matches using non-parametric tests.
